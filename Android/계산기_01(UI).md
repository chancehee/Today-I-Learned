### 2. UI 계산 결과 화면  
- 계산할 식을 보여주는 화면  
```xml
<TextView
        android:id="@+id/expressionTextView"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_marginStart="15dp"
        android:layout_marginTop="44dp"
        android:layout_marginEnd="15dp"
        android:gravity="end"
        android:text="1 * 2"
        android:textColor="@color/black"
        android:textSize="30sp"
        app:layout_constraintEnd_toEndOf="parent"
        app:layout_constraintTop_toTopOf="parent" />
```
- 계산한 결과를 보여주는 화면  
```xml
<TextView
        android:id="@+id/resultTextView"
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_marginStart="15dp"
        android:layout_marginEnd="15dp"
        android:gravity="end"
        android:text="2"
        android:textColor="#aaaaaa"
        android:textSize="20sp"
        app:layout_constraintBottom_toTopOf="@id/keypadLayout"
        app:layout_constraintEnd_toEndOf="parent"
        app:layout_constraintStart_toStartOf="parent" />
```

### 버튼: ShapeDrawable  
- 머티리얼 디자인 ripple을 통해 버튼 클릭시 효과,색상 지정
- 모양을 rectangle형태로 지정
- 모서리 부분을 둥글게 만들고, 테두리 색상 지정

```xml
<?xml version="1.0" encoding="utf-8"?>
<ripple xmlns:android="http://schemas.android.com/apk/res/android"
    android:color="@color/buttonPressGray">

    <item android:id="@android:id/background">
        <shape android:shape="rectangle">
            <solid android:color="@color/buttonGray"/>
            <corners android:radius="100dp"/>
            <stroke android:color="@color/buttonPressGray"
                android:width="1dp"/>
        </shape>
    </item>
</ripple>
```
